# TinyML Arduino Projects 🚀

This repository documents my journey of learning TinyML on microcontrollers.

## Hardware
- Arduino UNO Q

## Projects

### 01 - Sine Wave Basics
Display a sine wave on 8x13 LED matrix (no ML).

### 02 - TinyML Sine Wave Predictor
Train a neural network to approximate sin(x), convert to INT8, and deploy on Arduino.

## Goal
To combine Machine Learning + Embedded Systems for real-world TinyML applications.
