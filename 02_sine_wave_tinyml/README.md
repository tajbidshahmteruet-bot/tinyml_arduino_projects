# TinyML model training using DNN 🚀

This repository documents the training of models for fitting the sinewave using randomly created data, converting the model to TFlite format to deploy on microcontrollers, and result of training.

## Platform
- Python
- Tensorflow
- TensorflowLite
- numpy
- Matplotlib
